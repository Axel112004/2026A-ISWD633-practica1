Hasta este punto entendí lo básico de Docker: aprendí que una imagen es como una plantilla y que de ahí se crean los contenedores, que son los que realmente se ejecutan. También aprendí a descargar imágenes como nginx, rabbitmq y jenkins, y a crear contenedores usando docker run, además de mapear puertos para poder acceder desde el navegador. Vi cómo iniciar, detener y borrar contenedores, y la diferencia entre ejecutarlos en primer plano y en segundo plano. También aprendí a entrar dentro de un contenedor con docker exec -it para explorar archivos y usar comandos como ls, echo o cat, y a obtener datos importantes como la contraseña de Jenkins. Finalmente, entendí cómo usar docker logs para ver lo que está pasando dentro del contenedor y detectar errores.
